Stirling Numbers of the Second Kind

Count — and visualize every partition — of a set of n labeled elements into exactly k non-empty unlabeled subsets.

S(n, k) = k · S(n−1, k) + S(n−1, k−1)
S(0,0) = 1  ·  S(n,0) = 0  ·  S(0,k) = 0
A new element either joins one of the k existing subsets (k ways) or forms a new singleton, reducing the problem to S(n−1, k−1).
